# Contributing to EducaFree

Thank you for considering contributing to EducaFree! Here are some guidelines to get started:

## How to Contribute
- Fork the repository.
- Create a new branch (`git checkout -b feature-branch`).
- Make your changes.
- Commit your changes (`git commit -m 'Add new feature'`).
- Push to the branch (`git push origin feature-branch`).
- Open a pull request.
